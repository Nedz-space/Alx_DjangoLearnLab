### Book List Query Features

1. **Filtering**:
   - `/api/books/?title=MyBook`
   - `/api/books/?author__name=John%20Doe`
   - `/api/books/?publication_year=2024`

2. **Searching**:
   - `/api/books/?search=Python`

3. **Ordering**:
   - `/api/books/?ordering=title`
   - `/api/books/?ordering=-publication_year`
